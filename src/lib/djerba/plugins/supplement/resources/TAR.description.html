This assay combines two comprehensive next generation sequencing assays: a targeted sequencing (TAR) assay and a shallow whole genome sequencing (sWGS) assay.

TS libraries are prepared using the KAPA Hyper Prep kit with DNA extracted from FFPE, cfDNA, fresh frozen tissue (for tumour samples) or buffy coat blood specimens (for matched normal blood samples). Paired-end sequencing is performed using the NextSeq 550 technology. For targeted sequencing, alignments are performed using bwa mem (0.7.12) against reference genome GRCh38.p12. Unique molecule based error suppression is completed with ConsensusCruncher (https://github.com/pughlab/ConsensusCruncher). SNVs and INDELs are called using MuTect2 (GATK 4.1.1.0) and annotated with VariantEffectPredictor (105.0) followed by OncoKB. In addition, variants without an OncoKB actionable tier are reported if they are listed as oncogenic, likely oncogenic or predicted oncogenic.

Shallow whole genome sequencing libraries were prepared using the KAPA Hyper Prep kit with DNA extracted from FFPE, cfDNA, fresh frozen tissue (for tumour samples). Paired-end sequencing is performed using the NextSeq 550 technology to a minimum target coverage of 0.1x. Alignments are performed using bwa mem (0.7.12) against reference genome GRCh38.p12 and copy number amplifications are called using ichorCNA ichorCNA.
